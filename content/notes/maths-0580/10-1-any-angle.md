## Core idea

Trigonometric functions extend beyond the acute angles ($0^\circ$ and $90^\circ. 360^\circ$ coordinate grid. These values are mapped using trigonometric graphs, which display periodic wave patterns that repeat over fixed intervals. Use these trigonometric graphs to find multiple solution angles, analyze wave behaviors, and solve non-linear trigonometric equations.

## Key formulas

$$
\text{Sine Periodic Wave Properties:} \quad \sin(\theta) = \sin(180^\circ - \theta) = \sin(\theta + 360^\circ)

\text{Cosine Wave Symmetry Properties:} \quad \cos(\theta) = \cos(360^\circ - \theta) = \cos(\theta + 360^\circ)

\text{Tangent Periodic Wave Properties:} \quad \tan(\theta) = \tan(180^\circ + \theta) = \tan(\theta + 180^\circ)
$$

## Steps / method

Use your calculator to find the primary angle solution (the principal value) by applying the standard inverse trigonometric function ($\sin^{-1}, \cos^{-1}, \text{ or } \tan^{-1}$).

Sketch the relevant trigonometric wave graph ($\sin, \cos, \text{ or } \tan$ and $0^\circ \le \theta \le 360^\circ$.

Draw a horizontal reference line across your graph at the target value to locate all points where it intersects the trigonometric wave.

Apply the symmetry rules of the wave graph to calculate the secondary angle solutions within the domain.

## Examiner tip

A common mistake on Paper 4 is providing only the single angle solution given directly by your calculator. For example, solving $\sin(\theta) = 0.5$ for $0^\circ \le \theta \le 360^\circ$ and $30^\circ. 180^\circ - 30^\circ = 150^\circ$. Missing the second angle will cost you accuracy marks.

## Quick check

The sine and cosine graphs form smooth, continuous waves that oscillate between a maximum value of $+1$ and a minimum value of $-1$.

## Worked example — Example 1 (June 2022 P42 Q7b)

Solve the trigonometric equation $\cos(\theta) = -0.6$ for the domain range $0^\circ \le \theta \le 360^\circ$.

Use your calculator to find the primary angle solution using the inverse cosine function: $\theta_1 = \cos^{-1}(-0.6) \approx 126.9^\circ$.

Visualize or sketch a cosine graph wave. Notice that cosine is negative in the second and third quadrants ($90^\circ$ and $270^\circ$).

Apply the cosine wave symmetry rule to find the secondary $\theta_2 = 360^\circ - \theta_1$.

Calculate the secondary angle value: $\theta_2 = 360^\circ - 126.9^\circ = 233.1^\circ$.

State both valid solutions rounded to $1$ decimal place: $\theta = 126.9^\circ$. $233.1^\circ$.

## Worked example — Example 2 (Nov 2023 P41 Q9b)

Find all solutions to the equation $\tan(\theta) = 1.5$ within the domain range $0^\circ \le \theta \le 360^\circ$.

Find the primary solution using the inverse tangent function on your calculator: $\theta_1 = \tan^{-1}(1.5) \approx 56.3^\circ$.

Recall the periodic rule for the tangent function: the tangent wave repeats its pattern exactly every $180^\circ$.

Apply the tangent periodic rule to calculate the secondary $\theta_2 = \theta_1 + 180^\circ$.

Calculate the secondary angle value: $\theta_2 = 56.3^\circ + 180^\circ = 236.3^\circ$.

State both valid solutions rounded to $1$ decimal place: $\theta = 56.3^\circ$. $236.3^\circ$.
