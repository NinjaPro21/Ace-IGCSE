## Core idea

Substitution is the process of replacing algebraic variable letters with specific numerical values within an expression or formula. Once the variables are replaced, the expression is evaluated using the standard rules of arithmetic priority (BIDMAS). Use this technique whenever you need to compute target values from real-world equations, evaluate functions at given inputs, or verify algebraic solutions.

## Key formulas

$$
\text{Linear evaluation: } f(x) = mx + c \Rightarrow f(k) = mk + c
$$

$$
\text{Quadratic evaluation: } y = ax^2 + bx + c,\ x = -k \Rightarrow y = a(-k)^2 + b(-k) + c
$$

## Steps / method

Rewrite the given algebraic expression or formula completely, replacing every target variable letter with empty pairs of brackets.

Insert the specified numerical values into their corresponding brackets, paying careful attention to negative signs.

Apply the order of operations (BIDMAS) by expanding terms inside brackets and calculating powers/indices first.

Perform multiplications and divisions next, then finish the calculation with additions and subtractions to find the final numerical value.

## Examiner tip

When substituting a negative number into a squared variable, always enclose the number in brackets on your calculator. For example, evaluating $x^2 - 5x$ when $x = -3$ must be entered as $(-3)^2 - 5(-3)$, which equals $9 + 15 = 24$. Leaving out the brackets results in $-9 + 15 = 6$, which is a very common error that loses marks on Paper 2 and Paper 4.

## Quick check

If a negative value is substituted for a variable raised to an even power, that specific component always evaluates to a positive result.

## Worked example — Example 1 (May 2021 P21 Q4)

Given that $v = u + at$, calculate the value of $v$ when $u = -12$, $a = 3.5$, and $t = 8$.

Set up the formula by writing empty brackets for the variables: $v = (\ ) + (\ )(\ )$.

Substitute the given values into the brackets: $v = (-12) + (3.5)(8)$.

Perform the multiplication step before adding: $(3.5) \times 8 = 28$.

Combine the remaining values to find the final answer: $v = -12 + 28 = 16$.

## Worked example — Example 2 (Nov 2022 P41 Q2a)

Calculate the value of $W = \frac{3m^2 - 2k}{5}$ when $m = -4$ and $k = 7.5$.

Rewrite the fraction using brackets for variables $m$ and $k$: $W = \frac{3(\ )^2 - 2(\ )}{5}$.

Insert the values: $W = \frac{3(-4)^2 - 2(7.5)}{5}$.

Evaluate the index power first: $(-4)^2 = 16$, so the numerator becomes $3(16) - 2(7.5)$.

Perform both multiplications in the numerator: $3 \times 16 = 48$ and $2 \times 7.5 = 15$. Subtract: $48 - 15 = 33$.

Divide the numerator by the denominator to get the final answer: $W = \frac{33}{5} = 6.6$.
