## Core idea

Proportion models the mathematical relationship between two variables as they change relative to one another. Direct proportion means one variable increases by a constant multiple as the other increases, while inverse proportion means one variable decreases as the other increases. Use this framework to model physical laws, scale rates, and solve geometric growth problems by finding the constant of proportionality $k$.

## Key formulas

$$
y \propto x \Rightarrow y = kx
$$

$$
y \propto \frac{1}{x} \Rightarrow y = \frac{k}{x}
$$

$$
y \propto x^2 \Rightarrow y = kx^2
$$

$$
y \propto \sqrt{x} \Rightarrow y = k\sqrt{x}
$$

## Steps / method

Translate the statement of proportion into a formal equation using a constant of proportionality $k$ (e.g. convert "y is inversely proportional to $x$" into $y = \frac{k}{x}$).

Substitute the given initial values for both variables into the equation.

Solve this equation to find the exact numerical value of the constant $k$.

Rewrite the proportion formula using your calculated value for $k$, and use this equation to calculate unknown values when new inputs are provided.

## Examiner tip

Pay close attention to how the relationship is described. Questions often state that a variable is proportional to the square ($x^2$), cube ($x^3$), or square root ($\sqrt{x}$) of another variable, rather than just $x$. Missing these words leads to using the wrong equation structure, which will lose you all subsequent accuracy marks.

## Quick check

If a variable is inversely proportional to another, multiplying the two corresponding variable values together will always equal the constant value $k$.

## Worked example — Example 1 (June 2021 P22 Q12)

$y$ is directly proportional to the square of $x$. Given that $y = 18$ when $x = 3$, find the value of $y$ when $x = 5$.

Write the proportion statement as an equation with the constant $k$: $y = kx^2$.

Substitute the initial values $y = 18$ and $x = 3$ into the equation: $18 = k(3)^2$.

Evaluate the power and solve for $k$: $18 = 9k$, so $k = \frac{18}{9} = 2$.

Rewrite the complete formula using the found value of $k$: $y = 2x^2$.

Substitute the new input value $x = 5$ into this formula to find $y$: $y = 2(5)^2 = 2 \times 25 = 50$.

## Worked example — Example 2 (Nov 2023 P21 Q19)

$R$ is inversely proportional to the square root of $v$. Given that $R = 6$ when $v = 16$, find the value of $R$ when $v = 64$.

Write the inverse proportion statement as an equation with the constant $k$: $R = \frac{k}{\sqrt{v}}$.

Substitute the initial values $R = 6$ and $v = 16$ into the equation: $6 = \frac{k}{\sqrt{16}}$.

Calculate the square root and solve for $k$: $6 = \frac{k}{4}$, so $k = 6 \times 4 = 24$.

Rewrite the formula incorporating the constant value: $R = \frac{24}{\sqrt{v}}$.

Substitute the new input value $v = 64$ into the formula to find $R$: $R = \frac{24}{\sqrt{64}} = \frac{24}{8} = 3$.
