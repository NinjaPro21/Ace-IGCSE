## Core idea

In a right-angled triangle, the square on the hypotenuse equals the sum of the squares on the other two sides. Use this to find missing lengths in 2D and 3D shapes.

## Key formulas

$$
a^2 + b^2 = c^2 \quad (c \text{ is the hypotenuse})
$$

$$
c = \sqrt{a^2 + b^2} \qquad a = \sqrt{c^2 - b^2}
$$

## Steps / method

Find right-angled triangles in the diagram (draw extra lines if needed).

Label the hypotenuse — it is opposite the $90^\circ$ angle. 3. **Find hypotenuse:** square both shorter sides, add, then square root.

 4. **Find a shorter side:** square the hypotenuse, subtract the other shorter side squared, then square root.

## Examiner tip

In 3D questions you often use Pythagoras twice: first for a base diagonal, then for the final 3D length.

## Quick check

The hypotenuse is always the longest side in a right-angled triangle.

## Worked example — Example 1 (June 2022 P21 Q6)

A right-angled triangle has shorter sides $5\text{ cm}$ and $12\text{ cm}$. Find the hypotenuse. Write $a^2 + b^2 = c^2$ with $a=5$, $b=12$.

$5^2 + 12^2 = c^2 \Rightarrow 169 = c^2 \Rightarrow c = 13\text{ cm}$

## Worked example — Example 2 (Nov 2023 P42 Q3a)

A box has base $6\text{ cm} \times 8\text{ cm}$ and height $24\text{ cm}$. Find the space diagonal.

Base diagonal: $d^2 = 6^2 + 8^2 = 100 \Rightarrow d = 10\text{ cm}$.

Space diagonal: $D^2 = 10^{2} + 24^2 = 676 \Rightarrow D = 26\text{ cm}$.
