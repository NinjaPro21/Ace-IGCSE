## Core idea

Logarithms, powers, and standard-form calculations appear throughout 0580. Enter them with the correct keys and **brackets** so the calculator applies BIDMAS correctly.

## Steps / method

**Mission — log(100):** Press **log → 1 → 0 → 0 → =** on the diagram. Result must be **2**.

**Powers:** Base first, then **xʸ**, then exponent. For $e^x$, use **SHIFT + ln** (exp).

**Standard form:** Type **3.2 × 10ˣ 5** using the **×10ˣ** key — do not type `10` then use the power button separately.

**Reuse answers:** Press **Ans** instead of copying a rounded decimal by hand.

**Exact form:** Press **S⇔D** to toggle between fraction/surd and decimal before writing your final answer.

## Examiner tip

Wrap negative numbers in brackets before squaring: **(−5)^2 = 25** but **−5^2 = −25**. Give non-exact answers to **3 significant figures** unless told otherwise.

## Quick check

Which key toggles between exact fractions and decimals?
