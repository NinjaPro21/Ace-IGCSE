## Core idea

Indices (or exponents) determine the number of times a base number multiplies by itself. Fractional and negative powers extend this framework to represent roots and reciprocal structures. Use index laws to simplify products, quotients, and powers within complex algebraic transformations and fractional equations.

## Key formulas

$a^m \times a^n = a^{m+n}, \quad \frac{a^m}{a^n} = a^{m-n}, \quad (a^m)^n = a^{m \times n}$

$a^{-n} = \frac{1}{a^n}, \quad a^0 = 1, \quad a^{\frac{m}{n}} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m$

Index laws — multiply same base: add powers; divide: subtract powers; power of power: multiply powers.

## Steps / method

When multiplying terms with identical bases, add their exponents together while keeping the base number unchanged.

When dividing terms with identical bases, subtract the denominator exponent from the numerator exponent.

Convert negative exponents into fractions by placing the positive index counterpart in the denominator.

Evaluate fractional powers by finding the root specified by the denominator first, then raising the result to the power specified by the numerator.

## Examiner tip

A very common error on algebraic index questions is failing to apply the outside index to numerical coefficients inside the brackets. For example, expanding the expression $(3x^4)^3$ requires you to cube the coefficient $3$ as well as multiply the indices of the variable, which simplifies to $27x^{12}$, not $3x^{12}$.

## Quick check

Any non-zero base number raised to an index power of $0$ evaluates to exactly $1$.

## Worked example — Example 1 (June 2021 P22 Q18)

Simplify fully the expression: $(16x^8)^{\frac{3}{4}}$.

Distribute the fractional power to both components inside the brackets: $(16)^{\frac{3}{4}} \times (x^8)^{\frac{3}{4}}$.

Evaluate the numerical coefficient component: $(16)^{\frac{3}{4}} = (\sqrt[4]{16})^3$.

Compute the fourth root of $16$, which is 2, and cube it:$2^3 = 8$.

Simplify the variable index component using the power multiplication rule: $8 \times \frac{3}{4} = \frac{24}{4} = 6$, giving $x^6$.

Combine both simplified parts to state the final answer: $8x^6$.

## Worked example — Example 2 (Nov 2023 P21 Q9)

Solve the index equation for the variable value $x$. $3^{x-2} = \frac{1}{27}$.

Express both sides of the equation using a common prime base number, which is $3$.

Convert the right-hand side fraction into integer index form: $\frac{1}{27} = \frac{1}{3^3} = 3^{-3}$.

Equate both sides using this shared base layout: $3^{x-2} = 3^{-3}$.

Since the bases are identical, equate their exponents: $x - 2 = -3$.

Solve this linear equation by isolating $x$. So $x = -3 + 2 = -1$.
