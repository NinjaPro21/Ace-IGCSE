## Core idea

Quadratic equations are polynomial equations where the unknown variable has a maximum index power of 2. They can be solved analytically by factorising, completing the square, or using the quadratic formula. Use these methods to find the roots of quadratic equations, determine the turning points of curves, or find the $x$-intercepts of quadratic graphs.

## Key formulas

**Standard form**

$$
ax^2 + bx + c = 0
$$

**Quadratic formula**

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

**Completing the square**

$$
(x + p)^2 + q \quad \text{where } p = \frac{b}{2a} \text{ and } q = c - p^2
$$

## Steps / method

Rearrange the quadratic equation into the standard form $ax^2 + bx + c = 0$ before applying any solution method.

If the expression factorises easily, set each linear bracket factor to $0$ to find the roots.

For non-factorisable quadratics, use the quadratic formula.

Substitute the coefficients $a$, $b$, and $c$ into the formula, using brackets around negative values.

Calculate the two potential solutions separately using the $\pm$ operator, and round your answers to 2 decimal places unless specified otherwise.

## Examiner tip

Paper 4 questions often include instructions like "show all your working and give your answers correct to 2 decimal places." This wording is a clear hint that the quadratic equation will not factorise, and you should use the quadratic formula. Always state your substitutions into the formula explicitly to secure method marks before calculating.

## Quick check

If the value inside the square root of the quadratic formula ($b^2 - 4ac$) is negative, the equation has no real solutions.

## Worked example — Example 1 (June 2023 P41 Q3b)

Solve the equation $3x^2 - 5x - 4 = 0$, giving your answers to 2 decimal places.

Identify the coefficients for the quadratic formula: $a = 3$, $b = -5$, $c = -4$.

Write out the quadratic formula and substitute the coefficients carefully using brackets:

$$
x = \frac{-(-5) \pm \sqrt{(-5)^2 - 4(3)(-4)}}{2(3)}
$$

Simplify the terms inside the radical and the denominator: $x = \frac{5 \pm \sqrt{25 + 48}}{6}$, so $x = \frac{5 \pm \sqrt{73}}{6}$.

Compute the two solutions separately: $x_1 = \frac{5 + \sqrt{73}}{6} \approx 2.26$ and $x_2 = \frac{5 - \sqrt{73}}{6} \approx -0.59$.

## Worked example — Example 2 (Nov 2022 P22 Q11)

Express $x^2 + 6x - 5$ in the form $(x + p)^2 + q$, and use this to solve $x^2 + 6x - 5 = 0$.

Find the value of $p$ by halving the coefficient of $x$: $p = \frac{6}{2} = 3$.

Set up the squared bracket: $(x + 3)^2$. Expanding this gives $x^2 + 6x + 9$.

Adjust the constant term to match the original expression: we need to go from $+9$ to $-5$, so subtract $14$: $q = -5 - 3^2 = -14$.

Write the completed square form: $(x + 3)^2 - 14$. To solve the equation, set this form to zero: $(x + 3)^2 - 14 = 0$, so $(x + 3)^2 = 14$.

Take the square root of both sides, remembering to include both positive and negative roots: $x + 3 = \pm\sqrt{14}$.

Isolate $x$ to find the exact solutions: $x = -3 \pm\sqrt{14}$.
