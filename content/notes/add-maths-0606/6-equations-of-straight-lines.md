## Core idea

A straight line can be completely defined in a coordinate plane by knowing its slope and at least one point it passes through. Writing equations in distinct standard formats makes it easier to identify key characteristics, such as intercepts and directional vectors.

## Key formulas

$$
\text{Point-Gradient Form: } y - y_1 = m(x - x_1)
$$
$$
\text{Slope-Intercept Form: } y = mx + c
$$
$$
\text{General Linear Form: } ax + by + c = 0
$$

## Steps / method

Find the target line's gradient ($m$) using coordinate points or perpendicular relationships.

Substitute the gradient $m$ and the coordinates of a known point $(x_1, y_1)$ into the formula $y - y_1 = m(x - x_1)$.

Expand the bracket component and distribute the gradient value across both terms.

Rearrange the resulting terms into the specific format requested by the question (e.g., clearing fractions to match $ax + by + c = 0$).

### Key rule

Use $y - y_1 = m(x - x_1)$ when you know a point and gradient. Always rearrange to the form the question asks for.

## Worked example — Point-gradient form

Find the equation of the line through $(3, -2)$ with gradient $4$.

Substitute into $y - y_1 = m(x - x_1)$: $y - (-2) = 4(x - 3)$.

Simplify: $y + 2 = 4x - 12 \implies y = 4x - 14$.

## Worked example — General form

Write the equation of the line through $(0, 5)$ and $(4, 1)$ in the form $ax + by + c = 0$. Gradient: $m = \frac{1 - 5}{4 - 0} = -1$. Using $(0, 5)$: $y - 5 = -x \implies x + y - 5 = 0$.

## Examiner tip

Always read the final line of the question carefully to see what format is requested. If the examiner explicitly requests the answer in the form $ax + by + c = 0$ where a, b, c are integers, leaving it as $y = 2x + \frac{1}{3}$ will lose formatting marks.

## Quick check

If a line has a gradient of $m$ and passes through the origin, then its linear equation simplifies directly to $y = mx$.
