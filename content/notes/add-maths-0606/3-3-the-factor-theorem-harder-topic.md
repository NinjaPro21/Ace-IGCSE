## Core idea

The Factor Theorem is a shortcut: if $f(a) = 0$, then $(x - a)$ is a factor of $f(x)$. This lets you find factors by trial and error instead of long division.

## Key formulas

**Factor Theorem**

$$
f(a) = 0 \iff (x - a) \text{ is a factor of } f(x)
$$

**Linear factor form**

$$
f\!\left(\frac{b}{a}\right) = 0 \iff (ax - b) \text{ is a factor}
$$

## Steps / method

1. **Identify candidates** — test factors of the constant term (e.g. for constant 6, test $\pm 1, \pm 2, \pm 3$).

2. **Substitute** — calculate $f(a)$.

3. **Conclude** — if $f(a) = 0$, then $(x - a)$ is a factor.

### Key rule

For a factor $(2x - 1)$, test $x = \frac{1}{2}$, not $x = -\frac{1}{2}$. Set the bracket equal to zero first.

## Worked example — Finding a factor

Question: Show that $(x + 1)$ is a factor of $f(x) = x^3 - x^2 - 10x - 8$.

For $(x + 1)$, test $x = -1$:

$$
f(-1) = (-1)^3 - (-1)^2 - 10(-1) - 8 = -1 - 1 + 10 - 8 = 0
$$

Since $f(-1) = 0$, $(x + 1)$ is a factor.

## Worked example — Solving for $k$

Question: $(x - 2)$ is a factor of $x^3 + kx - 10$. Find $k$.

If $(x - 2)$ is a factor, $f(2) = 0$:

$$
8 + 2k - 10 = 0 \Rightarrow 2k = 2 \Rightarrow k = 1
$$

## Examiner tip

When testing for a factor like $(2x - 1)$, substitute $x = \frac{1}{2}$. Set the bracket to zero ($2x - 1 = 0$) to find the correct test value.

## Quick check

If $f(1) = 0$, then $(x - 1)$ is a factor. If $f(-1) = 0$, then $(x + 1)$ is a factor.
