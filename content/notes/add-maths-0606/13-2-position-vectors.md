## Core idea

Position vectors describe the location of a point relative to a fixed origin. Displacement between two points is found by subtracting position vectors, and vector geometry problems are solved by combining these displacements.

## Key formulas

$$
\text{Position vector of point } P: \mathbf{p} = \begin{pmatrix} x \ \\ y \end{pmatrix}
\vec{AB} = \mathbf{b} - \mathbf{a}, \quad |\mathbf{r}| = \sqrt{x^2 + y^2}
$$

## Steps / method

Write each point as a position vector from the origin.

Find $\vec{AB}$ using $\mathbf{b} - \mathbf{a}$.

Use Pythagoras on the components to find magnitude.

Express unknown points using given ratios along a line segment.

### Key rule

The position vector of the midpoint of $A$ and $B$ is $\frac{\mathbf{a} + \mathbf{b}}{2}$.

## Worked example — Position and displacement

$A$ has position vector $\begin{pmatrix} -1 \ \\ 4 \end{pmatrix}$ and $B$ has $\begin{pmatrix} 5 \ \\ 1 \end{pmatrix}$. Find $\vec{AB}$ and its magnitude.

$$
\vec{AB} = \begin{pmatrix} 5 \ \\ 1 \end{pmatrix} - \begin{pmatrix} -1 \ \\ 4 \end{pmatrix} = \begin{pmatrix} 6 \ \\ -3 \end{pmatrix}, \quad |\vec{AB}| = \sqrt{36 + 9} = 3\sqrt{5}
$$

## Worked example — Point dividing a line

$P$ divides $AB$ in the ratio $2 : 1$. Find $\mathbf{p}$ if $\mathbf{a} = \begin{pmatrix} 0 \ \\ 0 \end{pmatrix}$ and $\mathbf{b} = \begin{pmatrix} 6 \ \\ 3 \end{pmatrix}$.

$$
\mathbf{p} = \mathbf{a} + \frac{2}{3}\vec{AB} = \frac{2}{3}\begin{pmatrix} 6 \ \\ 3 \end{pmatrix} = \begin{pmatrix} 4 \ \\ 2 \end{pmatrix}
$$

## Examiner tip

Always subtract in the correct order: end minus start for displacement $\vec{AB}$.

## Quick check

If $\mathbf{a} = \mathbf{b}$, then $\vec{AB} = \mathbf{0}$.
