## Core idea

0606 uses **degrees** for geometry-style trig and **radians** for calculus. Your calculator must show the correct angle unit on the status line before you evaluate sin, cos, or tan.

## Steps / method

**Mission — Degree check:** On the interactive diagram, press **sin → 3 → 0 → =** in order. In **Degree** mode the display shows **0.5**.

If you see **−0.988…**, you are in **radians** — open **SHIFT + MENU SETUP → Angle Unit → Deg**.

**Mission — Radian check:** Switch to **Rad** in SETUP, then evaluate **sin 1**. You should see **0.841…** (not 0.5).

After clearing memory (**SHIFT + CLR → Setup Data**), re-run the sin 30 test in the mode the question requires.

## Examiner tip

Read the question carefully: geometry problems usually need **Deg**; calculus and small-angle work often need **Rad**. Re-check the status line after every mode switch.

## Quick check

What should **sin 30** equal in degree mode?

**Answer:** **0.5**
