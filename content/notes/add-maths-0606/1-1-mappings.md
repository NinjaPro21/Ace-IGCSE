## Core idea

Mappings describe how elements of one set (the domain) are related to elements of another set (the codomain). They categorize relationships as one-to-one, many-to-one, one-to-many, or many-to-many.

## Key formulas

Domain: The set of all possible input values (x). Range: The set of actual output values (y) produced by the domain.

## Steps / method

Identify Inputs: List the values in the domain.

Apply the Rule: Calculate the corresponding output for each input.

Classify Mapping: One-to-one: Each x has a unique y, and each y comes from exactly one x. Many-to-one: Multiple x values result in the same y value.

## Worked Example (0606/PYP Style)

Question: Given the mapping x↦x^2 for the domain {−2,−1,0,1,2}, identify the range and the type of mapping.

Apply the rule: (−2)$^2$=4,(−1)$^2$=1,0$^2$=0,1$^2$=1,2$^2$=4.

The range is {0,1,4}.

Since both −2 and 2 map to 4, this is a many-to-one mapping.

## Examiner tip

Be careful: the range is a subset of the codomain. It only includes values that are actually achieved by the function.

## Quick check

If a mapping is "one-to-many," then it is not a function.
