## Core idea

The graph of y=∣f(x)∣ where f(x) is a quadratic function represents a parabola whose negative vertical outputs are reflected vertically across the horizontal axis. Use this when analyzing non-negative structural functions where input directions can change but the output must remain positive. This transformation converts any sections below the x-axis into positive arches, creating sharp corners called cusps where the curve meets the x-axis.

## Key formulas

$$
f(x) = ax^2 + bx + c

y = |ax^2 + bx + c|
$$

## Steps / method

Sketch the standard base quadratic parabola $y = ax^2 + bx + c$ using a light pencil guide line, plotting its intercepts and turning point.

Locate the real roots ($x$-intercepts) where the parabola crosses the horizontal axis; if there are no $x$-intercepts, the graph requires no reflection.

Keep the portions of the parabola that lie above or on the $x$-axis completely unchanged.

Take the middle loop or outer segments that drop below the $x$-axis ($y < 0$) and reflect them vertically upward across the $x$-axis, erasing the old lower guide lines.

### Key rule

Sketch the base parabola first; reflect any portion below the $x$-axis upward, creating sharp cusps at each root.

## Worked example — Two roots, reflected loop

Sketch $y = |x^2 - 4x + 3|$.

Factorise: $(x - 1)(x - 3) = 0$, roots at $x = 1$ and $x = 3$. Vertex at $(2, -1)$.

Reflect the loop below the axis upward to form a local maximum at $(2, 1)$ with cusps at $(1, 0)$ and $(3, 0)$.

## Worked example — No reflection needed

Sketch $y = |x^2 + 1|$.

Since $x^2 + 1 \ge 1 > 0$ for all real $x$, the parabola never goes below the $x$-axis.

The modulus graph is identical to $y = x^2 + 1$.

## Examiner tip

When drawing reflected quadratic graphs in exams, make sure you don't round off the bottom of the reflected loops into smooth turning points. The graph must hit the x-axis at a sharp angle at each intercept, forming distinct points known as cusps. Smooth curves at these points will lose marks.

## Quick check

If a quadratic graph is completely reflected using a modulus operator, then any sharp point where it reflects off the x-axis is called a cusp.
