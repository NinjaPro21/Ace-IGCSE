## Core idea

Solving a quadratic inequality means finding the range of $x$ values that make the quadratic expression greater than or less than zero. This is almost impossible to do accurately without a quick sketch of the parabola.

## Key formulas

**Greater than zero (above x-axis)**

$$
ax^2 + bx + c > 0
$$

**Less than zero (below x-axis)**

$$
ax^2 + bx + c < 0
$$

## Steps / method

1. **Find critical values** by solving $ax^2 + bx + c = 0$.

2. **Sketch** the parabola using these roots.

3. **Identify the region**:
   - If $a > 0$ and you want $> 0$: answer is **outside** the roots ($x < \text{small}$ or $x > \text{big}$)
   - If $a > 0$ and you want $< 0$: answer is **between** the roots ($\text{small} < x < \text{big}$)

### Key rule

Never write $2 < x > 3$. Use **or** for two separate regions.

## Worked example — Positive leading coefficient

Question: Solve $x^2 - 5x + 6 > 0$.

Critical values: $(x - 2)(x - 3) = 0 \Rightarrow x = 2, 3$.

The parabola opens upward, crossing at $x = 2$ and $x = 3$.

The region where $y > 0$ is **outside** the roots:

$$
x < 2 \quad \text{or} \quad x > 3
$$

## Worked example — Negative leading coefficient

Question: Solve $4 - x^2 \ge 0$.

Critical values: $(2 - x)(2 + x) = 0 \Rightarrow x = \pm 2$.

The parabola is a "cap" shape opening downward.

The region where $y \ge 0$ is **between** the roots:

$$
-2 \le x \le 2
$$

## Examiner tip

Never write your answer as $2 < x > 3$. This is mathematically impossible. Use "or" for two separate regions.

## Quick check

If the inequality is $x^2 + 1 < 0$, there is no solution (the graph never goes below the axis).
