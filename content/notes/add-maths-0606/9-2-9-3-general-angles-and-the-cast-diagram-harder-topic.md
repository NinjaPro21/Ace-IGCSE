## Core idea

Trigonometry extends to angles of any size using the coordinate plane. For any angle $\theta$, find the **reference angle** $\alpha$ (acute angle to the x-axis), then use the **CAST diagram** to decide whether each ratio is positive or negative.

## Key formulas

**Reference angle**

$\alpha$ is the acute angle between the terminal arm and the x-axis.

**CAST diagram** (start in Q4, go anticlockwise)

Cos positive → All positive → Sin positive → Tan positive

**Exact values with sign**

$\sin\theta = \pm\sin\alpha$, $\cos\theta = \pm\cos\alpha$, $\tan\theta = \pm\tan\alpha$ — the sign comes from CAST.

## Steps / method

1. **Sketch** the angle and identify its quadrant.
2. **Find** the reference angle $\alpha$ measured from the x-axis.
3. **Use CAST** to determine the sign of the required ratio.
4. **Evaluate** using exact values for $\alpha$ ($30°$, $45°$, $60°$).

### Key rule

Always measure the reference angle from the **x-axis**, not the y-axis.

## Worked example — Exact value

Question: Find the exact value of $\sin 300°$.

$300°$ is in Q4. Reference angle: $\alpha = 360° - 300° = 60°$.

In Q4 only cos is positive, so $\sin 300° = -\sin 60° = -\frac{\sqrt{3}}{2}$.

## Worked example — Solving for $\theta$

Question: Find all $\theta$ for $0° \le \theta \le 360°$ given $\tan\theta = -1$.

Basic angle: $\alpha = \tan^{-1}(1) = 45°$. Tan is negative in Q2 and Q4.

$\theta = 180° - 45° = 135°$ or $\theta = 360° - 45° = 315°$.

## Examiner tip

Measure reference angles from the horizontal x-axis ($0°$ or $180°$), not the vertical y-axis ($90°$ or $270°$). Measuring from the y-axis gives the wrong ratio entirely.

## Quick check

$200°$ lies in Q3 where tan is positive, so $\tan 200° > 0$.
