## Core idea

A function is a specific mapping where every input in the domain has exactly one output. Only one-to-one and many-to-one mappings are functions.

## Key formulas

f(x)=… or f:x↦…

Vertical Line Test: A curve is a function if every vertical line crosses it at most once.

## Steps / method

Check Mapping: Ensure no input produces multiple outputs.

Verify Domain: Every element in the specified domain must have a valid image.

## Worked Example (0606/PYP Style)

Question: Explain why f(x)=± x is not a function.

For a single input (e.g., x=4), there are two outputs (2 and −2).

This is a one-to-many mapping, which fails the definition of a function.

## Examiner tip

In the exam, if asked why a mapping isn't a function, use the technical term: "It is a one-to-many mapping".

## Quick check

If a horizontal line crosses a function's graph more than once, the function is many-to-one.
