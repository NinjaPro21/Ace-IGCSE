## Core idea

Trigonometric functions extend beyond right-angled triangles to encompass any angle of rotation by using the coordinates of a point moving around a unit circle. The sign of these ratios across the four quadrants is determined by the ASTC rule (All, Sine, Tangent, Cosine), which generates periodic wave patterns when graphed.

## Key formulas

$$\text{ASTC Rule: Q1 (All), Q2 (Sine), Q3 (Tangent), Q4 (Cosine)}$$

$$\text{Base Periods: } y = \sin x \, (360^\circ / 2\pi), \quad y = \cos x \, (360^\circ / 2\pi), \quad y = \tan x \, (180^\circ / \pi)$$

$$\text{Transformed Period for } \sin(bx) = \frac{360^\circ}{|b|} \text{ or } \frac{2\pi}{|b|}$$

## Steps / method

Locate which quadrant the given angle $\theta$ lies in (e.g., $135^\circ$ is in Quadrant 2).

Calculate the acute reference angle $\alpha$ measured directly to the horizontal x-axis ($\alpha = 180^\circ - 135^\circ = 45^\circ$).

Find the value of the trigonometric ratio for the reference angle $\alpha$.

Apply the ASTC quadrant rule to determine whether the final value should be positive or negative (Sine is positive in Q2, so $\sin 135^\circ = +\sin 45^\circ$).

Sketch wave transformations by altering amplitude ($a$ in $a\sin x$) or frequency ($b$ in $\sin bx$).

### Key rule

Use ASTC to fix the sign: Q1 (All +), Q2 (Sin +), Q3 (Tan +), Q4 (Cos +). Find the reference angle $\alpha$ from the x-axis first.

## Worked example — Exact value in Q2

Find the exact value of $\sin 150^\circ$.

$150^\circ$ is in Q2. Reference angle $\alpha = 180^\circ - 150^\circ = 30^\circ$.

Sine is positive in Q2, so $\sin 150^\circ = +\sin 30^\circ = \frac{1}{2}$.

## Worked example — Graph period

Find the period of $y = \sin 3x$ in radians.

$$\text{Period} = \frac{2\pi}{|b|} = \frac{2\pi}{3}$$

## Examiner tip

When graphing trigonometric functions, always label your key boundary axis points clearly (peaks, troughs, and intercepts at $90^\circ, 180^\circ, 270^\circ, 360^\circ$). Missing these axis labels can result in losing full accuracy marks on graph-drawing questions.

## Quick check

If an angle lies in the third quadrant (between $180^\circ$ and $270^\circ$), then its tangent ratio is positive, while its sine and cosine ratios are negative.
