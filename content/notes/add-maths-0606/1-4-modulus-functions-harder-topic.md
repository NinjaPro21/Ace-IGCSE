## Core idea

The modulus $|x|$ represents the absolute magnitude of $x$, ignoring its sign. It is used to measure distance from zero and to ensure non-negative results.

## Key formulas

**Definition of modulus**

$$
|x| = \begin{cases}x & \ \text{if } x \ge 0 \ -x & \text{if } x < 0\end{cases}
$$

**Modulus equation**

$$
|f(x)| = g(x) \Rightarrow f(x) = g(x) \quad \text{or} \quad f(x) = -g(x)
$$

## Steps / method

1. **Split the equation** — write two linear (or simpler) equations from the positive and negative cases.

2. **Solve** — find $x$ for both cases.

3. **Validate** — substitute each answer back into the original equation and reject any extraneous solutions.

### Key rule

A modulus can never equal a negative value. Always check solutions when the right-hand side contains $x$.

## Worked example — Simple equation

Question: Solve $|2x - 5| = 3$.

**Case 1:** $2x - 5 = 3 \Rightarrow 2x = 8 \Rightarrow x = 4$

**Case 2:** $2x - 5 = -3 \Rightarrow 2x = 2 \Rightarrow x = 1$

Both solutions are valid.

## Worked example — Equation with variable on both sides

Question: Solve $|x + 2| = 2x - 1$.

**Case 1:** $x + 2 = 2x - 1 \Rightarrow x = 3$

**Case 2:** $x + 2 = -(2x - 1) \Rightarrow x + 2 = -2x + 1 \Rightarrow 3x = -1 \Rightarrow x = -\frac{1}{3}$

Check $x = -\frac{1}{3}$: the right side is $2(-\frac{1}{3}) - 1 = -\frac{5}{3}$, which is negative, so this solution is rejected.

The only solution is $x = 3$.

## Examiner tip

Always check for extraneous solutions when the non-modulus side of the equation contains a variable. A modulus can never equal a negative value.

## Quick check

If $|f(x)| = -2$, there are no real solutions.
