## Core idea

An inverse function $f^{-1}(x)$ reverses the operations of $f(x)$, mapping outputs back to their original inputs. An inverse exists if and only if $f(x)$ is **one-to-one**. Graphically, the curve of $f^{-1}(x)$ is a reflection of $f(x)$ across the line $y = x$.

## Key formulas

$$f(f^{-1}(x)) = x \quad \text{and} \quad f^{-1}(f(x)) = x$$

$$\text{Domain of } f^{-1} = \text{Range of } f, \quad \text{Range of } f^{-1} = \text{Domain of } f$$

## Steps / method

Set the function equation to a temporary variable: $y = f(x)$.

Rearrange the equation algebraically to isolate $x$ explicitly in terms of $y$.

Swap the variables $y$ and $x$ to state the inverse: $f^{-1}(x) = \ldots$

State the domain of $f^{-1}(x)$ using the range of the original function $f(x)$.

### Key rule

The final answer must be written as $f^{-1}(x) = \ldots$ using $x$ as the independent variable — never leave the answer in terms of $y$.

## Worked example — Finding the inverse

Find $f^{-1}(x)$ for $f(x) = 2x + 3$, and state its domain.

Let $y = 2x + 3$. Rearrange: $x = \frac{y - 3}{2}$.

Swap variables: $f^{-1}(x) = \frac{x - 3}{2}$.

Domain of $f^{-1}$ = range of $f$ = $\mathbb{R}$.

## Worked example — Reflection in $y = x$

If $f(2) = 7$, what point lies on $f^{-1}(x)$?

Since $(2, 7)$ is on $f$, the reflected point $(7, 2)$ lies on $f^{-1}$.

Check: $f^{-1}(7) = \frac{7-3}{2} = 2$ ✓

## Examiner tip

Students often lose marks by writing the final inverse in terms of $y$ instead of $x$. Always express the answer clearly as $f^{-1}(x) = \ldots$

If $f(x)$ is not one-to-one over its full domain, restrict the domain before finding the inverse.

## Quick check

If the graph of $f(x)$ passes through $(a, b)$, then the graph of $f^{-1}(x)$ must pass through $(b, a)$.
