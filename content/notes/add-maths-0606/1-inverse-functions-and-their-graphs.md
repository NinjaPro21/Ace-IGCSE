## Core idea

An inverse function $f^{-1}(x)$ reverses the operations of $f(x)$, mapping the output elements back to their original inputs. An inverse function exists if and only if $f(x)$ is a one-to-one function; graphically, the curve of $f^{-1}(x)$ is a perfect reflection of $f(x)$ across the line $y = x$.

## Key formulas

$$f(f^{-1}(x)) = x \quad \text{and} \quad f^{-1}(f(x)) = x$$

$$\text{Domain of } f^{-1} = \text{Range of } f, \quad \text{Range of } f^{-1} = \text{Domain of } f$$

## Steps / method

Set the function equation to a temporary variable: $y = f(x)$.

Rearrange the equation completely using algebraic operations to isolate $x$ explicitly in terms of $y$.

Swap the variables $y$ and $x$ to state the final inverse function formula: $f^{-1}(x) = \dots$

State the domain of $f^{-1}(x)$ by calculating or reading the range of the original function $f(x)$.

## Examiner tip

Students often lose marks by writing the final inverse function in terms of $y$ instead of $x$. Always ensure your final answer is expressed clearly as $f^{-1}(x) = \dots$ using $x$ as the independent variable.

## Quick check

If the graph of $f(x)$ intersects the line $y = x$ at a point, then the graph of $f^{-1}(x)$ must pass through that exact same point.
