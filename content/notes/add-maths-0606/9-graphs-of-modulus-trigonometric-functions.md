## Core idea

Graphing the modulus of a trigonometric function means reflecting any negative parts of the wave (the troughs below the horizontal axis) upwards into the positive territory. This transformation changes the symmetry of the graph, often halving its period and turning smooth wave troughs into sharp, upward-pointing crests.

## Key formulas

$$y = |\sin x| \quad (\text{Period halves from } 360^\circ \text{ to } 180^\circ)$$

$$y = |a\cos(bx) + c|$$

## Steps / method

Sketch the regular, un-moduled trigonometric function $y = a\cos(bx) + c$ lightly using a dashed guide line.

Identify all the zero-crossing intercept points where the wave passes through the horizontal x-axis.

Leave the positive segments of the wave that sit above the x-axis untouched.

Flip the negative wave loops located below the x-axis vertically upwards, making sure the new peaks reach the same relative height as the original troughs.

## Examiner tip

Pay close attention when the function contains a vertical shift, like $y = |\sin x - 1|$. Calculate the value inside the modulus first before drawing the reflection, as shifting the graph vertically changes which sections fall below the x-axis.

## Quick check

If $y = |\cos x|$, the output range is compressed entirely between a minimum of 0 and a maximum of 1.
