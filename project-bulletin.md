# AceIGCSE V2: Master Bulletin Board

## 🎯 The Core Mission
A premium, gamified, mastery-based IGCSE learning platform focused on interactivity and clarity.

## 🧭 Feature Requirements
* **Khan Academy Mastery Loop:** Level 0 (Notes) -> Level 1 (Easy) -> Level 2 (Medium) -> Level 3 (Hard) -> Level 4 (PYP Mastery).
* **XP Engine:** Global Experience Points tracker and daily streaks.
* **Progress Dashboard:** Visual meters and badges for each chapter.
* **Data Tracking:** LocalStorage for user state; aggregate analytics for EC metrics.

## 🛠️ System Architecture
* `index.html`: Main shell.
* `src/NavigationController.js`: View routing and screen clearing.
* `src/MasteryEngine.js`: Progress math and state saving.
* `src/QuizArena.js`: Dynamic quiz logic.
* `src/ContentRenderer.js`: Dynamic note card rendering.

## 🎓 University Application Strategy
* **Focus:** Data-driven development, scaling curriculum modules, and user engagement analytics.
