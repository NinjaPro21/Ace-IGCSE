# AceIGCSE UI/UX & CSS Blueprint

## 1. Color Palette & Backgrounds
* **Primary App Background:** Soft cream/off-white (#FAF6F0).
* **Card Backgrounds:** Solid white (#FFFFFF).
* **Border Accents:** Thin, subtle borders (1px solid #E6DFD5).
* **Status Tags:** Light mint green (#E2F7ED) with dark green text (#1E5236).

## 2. Card Layouts & Grid Systems
* **Structure:** Uniform grid cards with 24px gap.
* **Border Radii:** 12px or 16px for all containers.
* **Top-Border Accents:** 4px thick lines in pastel colors (blue, orange, lavender) to differentiate topics.
* **Hover States:** Smooth scale-up transition (0.2s ease-in-out).

## 3. Premium Typography
* **Headers:** High-contrast Serif (e.g., Playfair Display or Georgia). Dark charcoal (#1A1A1A).
* **Body:** Clean Sans-Serif (e.g., Inter or Roboto). Soft charcoal (#4A4A4A).
* **Metadata:** Small, uppercase bold gold/tan (#B59A73) for chapter titles.

## 4. UI Layouts & Comparison Blocks
* **Side-by-Side Cards:** Horizontal layouts for contrasting concepts.
* **Left-Border Accents:** 4px vertical lines in vibrant colors for specific strategies.
* **Numbered Lists:** Light-gray rounded tags for step-by-step instructions.

## 5. Math & Interactive Elements
* **Interactive Graphs:** Dark-navy backgrounds (#111625) with neon plot lines.
* **Slider Controls:** Low-profile tracks with rounded thumb points and numeric boxes.
* **Math Rendering:** Clean LaTeX integration via MathJax/KaTeX.
